# 🧙‍♂️ Alchemist's Grimoire - Medicine Tracker

A mystical-themed **medicine tracking web app** that helps users manage pills, track doses, and visualize medication habits — all wrapped in a magical *grimoire-inspired UI*.

---

## ✨ Features

### 🔐 Authentication (Simulated)

* Login / Signup / Password Reset UI
* Input validation and error handling
* Smooth UI transitions

### 💊 Pill Management

* Add, edit, and delete medicines
* Track:

  * Name
  * Dosage
  * Time
  * Frequency

### ⏰ Smart Reminder System

* Auto-calculates dose timings
* Displays pending doses
* Mark doses as:

  * ✅ Taken
  * ❌ Missed

### 📊 Analytics Dashboard

* View:

  * Taken doses
  * Missed doses
* Visual progress bar chart
* Tracks total medication adherence

### 🎨 UI & Experience

* Dark fantasy “alchemy grimoire” theme
* Fully responsive design
* Tab-based navigation system

## ⚠️ Limitations

* No real authentication (UI only)
* Data stored per browser (not cloud synced)
* Reminders work only when app is open
